# Sut Note to MP

基于 [note-to-mp](https://github.com/sunbooshi/note-to-mp) 二次开发的 Obsidian 插件，适配个人使用需求。

## 功能

- 将 Obsidian 笔记发送到微信公众号草稿箱
- 添加更多适合自己，排版更加合理的样式
- 复制笔记内容到微信公众号编辑器，保留笔记样式
- 支持代码高亮、代码行号
- 支持本地图片上传
- 自定义主题样式

## 与原项目的差异

- 调整了部分默认配置
- 移除/修改了不适用的功能
